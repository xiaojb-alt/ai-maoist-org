# 参与贡献

感谢你有兴趣参与这个项目！

## 这不是一个"看代码"的项目

本项目目前是**理论框架阶段**，没有可运行的代码仓库。核心资产是一套方法论——如何用毛选思想治理 AI Agent 系统。

## 你可以怎么参与

### 1. 提 Issue
- **理论漏洞**：框架哪个环节有逻辑缺陷？
- **场景补充**：你遇到过的 Agent 组织熵增案例
- **工程落地**：某个机制已经有了具体实现方案

### 2. 开 Discussion
- 分享你在实际 Agent 系统中遇到的问题
- 讨论政委 Agent 的 Prompt 设计
- 分享 Token 预算的实践经验

### 3. 提 PR
- 补充缺失的文档板块
- 提供参考实现（Python/TypeScript 代码）
- 修正翻译或术语不准确的地方

## 讨论方向

当前最需要讨论的问题：

- 政委 Agent 的审核标准如何量化？
- 红蓝对抗在成本有限的情况下如何抽样？
- 群众反馈中的噪声如何过滤？
- Token 预算算法有没有更好的模型？

## 行为准则

保持开放、尊重的讨论氛围。这是一个探索性的理论框架，欢迎批评，拒绝人身攻击。

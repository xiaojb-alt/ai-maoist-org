# Contributing

Thanks for your interest in this project!

## This is not a "read the code" project

We are currently in the **theory-framework stage** — there is no runnable codebase yet. The core asset is a methodology: applying Maoist organizational theory to AI Agent system governance.

## How to Contribute

### 1. Open an Issue

- **Theory flaws**: Is there a logical gap in the framework?
- **Use case additions**: Share cases of Agent entropy you've encountered
- **Engineering implementations**: You have a concrete implementation plan for a mechanism

### 2. Start a Discussion

- Share problems you've faced in real Agent systems
- Discuss Political Commissar Agent prompt design
- Share practical experience with token budgets

### 3. Submit a PR

- Fill in missing documentation sections
- Provide reference implementations (Python / TypeScript)
- Fix inaccurate translations or terminology

## Discussion Directions

The topics most in need of discussion right now:

- How to quantify Political Commissar Agent review criteria?
- How to sample Red-Blue confrontations under cost constraints?
- How to filter noise in public feedback?
- Are there better models for token budget algorithms?

## Code of Conduct

Keep the discussion open and respectful. This is an exploratory theoretical framework. Criticism is welcome; personal attacks are not.
